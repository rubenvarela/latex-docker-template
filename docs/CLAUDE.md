# docs/ - Documentation

This directory contains user documentation for the LaTeX template repository.

## Purpose

Provide comprehensive guides for users to understand, use, and customize the template.

## Files

| File | Description |
|------|-------------|
| `01-getting-started.md` | Quick start guide and prerequisites |
| `02-project-structure.md` | Directory layout and file organization |
| `03-latex-packages.md` | All included packages and usage |
| `04-building.md` | How to compile documents |
| `05-testing.md` | Testing setup and CI/CD |
| `06-customization.md` | How to extend and customize |

## Conventions

- **Numbered prefixes**: Files are numbered for reading order
- **Lowercase with hyphens**: Use `topic-name.md` format
- **Markdown format**: GitHub-flavored markdown
- **Code examples**: Include runnable examples where possible

## Adding Documentation

1. Create a new file with the next number prefix
2. Use descriptive name: `07-new-topic.md`
3. Follow the existing format:
   - Title as H1
   - Clear sections with H2/H3
   - Code blocks for examples
4. Update this CLAUDE.md if adding major sections

## Style Guide

- Keep explanations concise
- Use bullet points for lists
- Include code examples
- Link to related documentation
- Use tables for structured information
